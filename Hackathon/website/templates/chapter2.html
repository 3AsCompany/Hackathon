Chapter 2 Linear Equations in one Variable

Question 1

Which of the following is not a linear equation in one variable?

(a) 33z+5 = 0

(b) 33(x+y) = 0

(c) 33x+5 = 0

(d) 33y+5 = 0


Answer: (b)

Question 2

The solution of 2x-3=7 is:

(a) 5

(b) 7

(c) 12

(d) 11


Answer: (a)

Question 3
